# 🐍 2d-snake-game

A classic 2D Snake Game built using Java Swing.

## 🎮 Features

- Snake movement using arrow keys
- Food spawning system
- Live score tracking
- Game over screen
- Restart button
- Collision detection

## 🛠 Technologies Used

- Java
- Java Swing
- OOP Concepts
- Event Handling
